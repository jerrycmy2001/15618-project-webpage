3D Object Rasterization and Rendering with CUDA

Minyu Cai, Jinyan Zhu

Introduction

We implemented 3D object rasterization to render a 360-degree camera view of the object. We demonstrated how we can achieve great speedup by parallelizing the task among multiple GPUs with CUDA and OpenMP.


Parallel Algorithms

- Projection: project vertices of triangles with respect to camera view

- Rasterization: determine the color of each pixel in the image given triangles
